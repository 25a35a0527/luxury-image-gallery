# Luxury Image Gallery

A modern and responsive image gallery website built using HTML and CSS with stylish layouts, hover animations, smooth transitions, and premium visual design.

## 🚀 Features

- Responsive Gallery Layout
- Modern UI Design
- Smooth Hover Effects
- Image Zoom Animations
- Interactive Gallery Cards
- Clean & Elegant Design
- Mobile Friendly Interface
- Smooth CSS Transitions

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design
- CSS Animations

## 📱 Responsive Design

This project is optimized for:
- Desktop Devices
- Tablets
- Mobile Phones

## 🎯 Project Objective

The objective of this project is to create a visually attractive and responsive image gallery using modern frontend design techniques and CSS animations.
